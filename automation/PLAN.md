# 크라마 플랜 v2 — 진짜 가치 × 재미 (2026-07-05)

> 구 로드맵(측정·유통 중심)은 폐기. 완료분은 흡수: GA4 계측·알림 넛지·스트릭 칩·원탭 세금알림·내부링크·FAQ 스키마·네이버 등록.
> **원칙**: ①사용자가 "오늘 이걸 하러 크라마 간다"고 말할 수 있는 것만 만든다 ②우리가 이미 가진 자산(밀웜·크라미·프로필 6축·알림·실데이터)을 재조합한다 ③재미는 금융과 붙어야 한다(게임을 위한 게임 금지).

## P0 — 간판급 2개

### ① 오르까 내리까 — 주간 금융 예측 게임 🎮
**뭐냐**: 매주 "이번 주 코스피/환율/기준금리, 오를까 내릴까?" 3문항 예측 → 마감 후 실제 결과로 채점 → 맞히면 밀웜, 연속 적중 배지.
**왜 가치**: 예측하려면 한 번 생각해야 함 = 가장 가벼운 금융 학습. 결과 확인하러 무조건 재방문(주 2회 방문 훅). 밀웜 경제에 '버는 재미' 추가.
**있는 것 재사용**: vote 시스템(빼꼼 투표)·밀웜 지급 RPC·업적 시스템·브리핑(결과 발표 코너로 연결).
**만들 것**: predictions 테이블(문항·마감·정답·응답), /play/ 페이지, 채점 크론(sync-calendar에 편승, 환율·지수는 공개 API), 브리핑에 "이번 주 예측 현황" 위젯.
**지표**: 주간 참여자, 참여자 D7.

### ② 내 돈 리포트 — 프로필 기반 연간 요약 📊
**뭐냐**: 프로필 6축(나이·지역·연봉·가구·차·집)으로 "올해 당신: 세금 약 X만원(재산세+자동차세+소득세) · 놓치고 있을 혜택 Y건 Z만원 · 중위소득 N%" 원페이지 리포트 + 공유 카드.
**왜 가치**: 흩어진 계산기 7종의 결과를 "나"로 묶는 킬러 개인화. 프로필 등록의 보상이 명확해짐 → 프로필 등록률 상승 → 전체 사이클 강화. 공유 카드는 유통까지 담당.
**있는 것 재사용**: 계산 로직 전부(salary·car-tax·property-tax·median-income), benefits 매칭, 공유 카드 캔버스(benefits에 있음), CramaProfile.
**만들 것**: /my-report/ 페이지(클라이언트 계산), 미등록 축은 "+ 추가하면 더 정확" 넛지, 1080×1350 공유 카드.
**지표**: 리포트 조회→프로필 축 추가 전환, 공유 클릭.

## P1 — 재미 루프 3개

### ③ 7일 챌린지 (무지출/만원의 하루) 🔥
매일 체크인(성공/실패 셀프 체크) → 7일 완주 시 밀웜 크게+전용 배지. 스트릭 UI·업적 시스템 재사용. 시즌제(월 1회 오픈)로 희소성.

### ④ 내 연봉 상위 몇 %? 🏆
/salary 상세에 "동갑·전체 중 상위 N%" (통계청 임금근로 일자리 소득 분포 — 공개 통계로 분위표 상수화). 재미+공유욕 자극, pSEO 가치도 큼. 프로필 나이 연동.

### ⑤ 밀웜 뽑기 (주 1회) 🎰
밀웜 소액으로 참여하는 옷장 아이템 뽑기(중복 시 밀웜 환급). 밀웜 사용처 확대 = 버는 재미(①③)와 쓰는 재미의 순환. 기존 옷장 아이템 재사용, 신규 아트 불필요.
